Welcome to YourCall

A unified platform combining video sharing with community communication. Upload videos, join servers, chat with friends, and discover amazing content - all in one place.

View on GitHub Documentation

✨ Features

📹 Video Sharing

Upload and stream MP4/WebM videos. Organize your content in a beautiful library with likes, comments, and shares.

🗣️ Community Servers

Create dedicated servers for your community. Manage channels, set roles, and build your audience.

💬 Real-time Chat

Live text chat with synchronized video playback. Watch together with live overlays and notifications.

🎤 Voice Calls

Crystal-clear voice communication. Connect with your community in real-time.

🛡️ Moderation Tools

Comprehensive moderation suite with user management, permissions, and content filtering.

📱 Cross-Platform

Web app, mobile responsive, and Android APK support. YourCall everywhere.
